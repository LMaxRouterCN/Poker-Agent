# Poker Agent
>*Fake Agent*

这是一个利用现在LLM网页端免费额度实现的完全免费的低配版Agent

流程:
- 安装油猴脚本`PokerAgent.js`
- 启动本地服务`agent_gui.py`,建议python版本3.14
- 把提示词和需求发送给网页端llm

llm输出指令 > 油猴脚本抓取指令 > 发送给本地服务 > 本地服务执行 > 本地服务发送回执给油猴脚本 > 油猴脚本将回执输入进输入框并发送 > llm收到回执并开始下一步工作

- 目前建议全程人工值守,因为如果出了报错llm又不知道哪错了他就会开始瞎猜,然后发生提示词污染导致那次会话基本废了,而且即使不废也会浪费上下文,我在考虑加入重试次数机制,llm发出未知指令超过几次就发一遍指令列表
- 目前已经可用,但还存在一些问题
- 默认适配chatglm.cn
- 因为llm会遗忘早期对话,考虑在固定的对话次数后重新发送一遍提示词
- 以越狱方式的"系统提示词"其实并不好,他会抹杀llm的个性以及其他的一些东西,会变傻,建议以"用户的额外要求"或者"使用指南"的方式来
- llm会瞎猜指令,没办法解决,如果llm在第一次对话时拿到了指令列表,之后过100次对话后即使事实上llm已经把指令列表忘干净了,他也依然不会再看一遍指令列表,而是会选择瞎猜
